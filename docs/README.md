# VIDA360 Documentation

## Overview
VIDA360 is a multi-tenant personal productivity app integrating Tasks, Habits, Goals, Finance, and Calendar.

- [TECHNICAL_HANDOFF_REPORT.md](./TECHNICAL_HANDOFF_REPORT.md) - Project overview and handoff

## Tech Stack
- **Frontend**: React + Vite + TypeScript + Tailwind CSS
- **Backend**: Supabase (Auth + PostgreSQL + RLS)
- **State**: TanStack Query + React Context
